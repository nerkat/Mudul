# Showcase Screens

Store product showcase screenshots in this folder.

Suggested filenames for the screenshots attached in chat:

- `01-client-dashboard-acme-corp.png`
- `02-org-dashboard-workspace.png`
- `03-new-call-form.png`
- `04-sales-call-dashboard-follow-up.png`

Recommended layout:

- `docs/showcase-screens/README.md` — manifest and naming guide
- `docs/showcase-screens/raw/` — original screenshots from chat or browser capture

Notes:

- The current chat attachment images are visible to the agent, but their binary files are not directly writable into the workspace through the available tools.
- If you drag those screenshots into `docs/showcase-screens/raw/` using VS Code Explorer, they will be included in the project in the intended location.
